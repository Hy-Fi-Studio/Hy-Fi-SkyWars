# Hy-Fi-SkyWars
Hy-Fi SkyWars offers a polished SkyWars implementation for Hytale servers. Players spawn on islands, loot chests, bridge, and fight until one remains. Supports solos, doubles, ranked modes, chest tiers, cage spawns, mid-events, and map queues. Perfect for PvP hubs and lobby networks.
